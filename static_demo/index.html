Hashtag Selector

This is the Hashtag Selector Page. It allows you to select initial hashtags that might be relevent to the first responder community you are looking for.
The list is pre populated with 17 hashtags that have been determined to cover a wide range of first responder communities.


Tweet Activity Timeline Chart

This is a chart that shows tweet activity over time for the tweets that contain hashtags which were selected from the Hashtag Selector Page.
This chart allows you to zoom in and select the start and end times for which you want to view tweets.


Topic Breakdown of Hashtags Pie Chart

This chart shows the 10 most probabalistic topics for the tweets specified by the start time and the end time from the Tweet Activity Timeline Chart.
Clicking on a topic (pie slice) will show you the top hashtags in the topic.


User - Hashtag Cooccurence Matrix

This matrix shows how often users post on hashtags specified from the topic selected above.
Moving your mouse over a cell will highlite the user, hashtag, and a tooltip will tell you the frequency that user mentioned that hashtag.
The color of each cell is proportional to the relative frequency that the user mentioned that hashtag.
